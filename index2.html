// Cara Kerja Algoritma
  • 1
    Ubah setiap exchange rate r menjadi bobot edge: w = −log(r)
  • 2
    Inisialisasi dist[src] = 0, semua node lain = ∞
  • 3
    Lakukan V−1 iterasi relaxasi semua edge
  • 4
    Iterasi ke-V: jika masih ada edge yang bisa di-relax → negative cycle ditemukan = ARBITRASE!
  • 5
    Trace back cycle untuk menemukan path konversi yang menguntungkan
O(V · E) = O(n² · n) = O(n³)
